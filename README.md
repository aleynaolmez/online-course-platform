# Online Kurs Platformu

Bu proje, İnternet Programlama dersi kapsamında geliştirilmiş bir online kurs platformudur.

## Kullanılan Teknolojiler
- PHP
- MySQL
- Bootstrap
- HTML / CSS / JavaScript
- XAMPP

## Özellikler
- Admin, eğitmen ve öğrenci rolleri
- Kurs ekleme, listeleme ve kayıt olma
- Kullanıcı yönetimi
- Yorum ve değerlendirme sistemi

## Kurulum
1. Proje XAMPP `htdocs` klasörüne kopyalanır
2. MySQL üzerinde veritabanı oluşturulur
3. `localhost` üzerinden çalıştırılır
